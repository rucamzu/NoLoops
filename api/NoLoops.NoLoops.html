Do<TSource>(IEnumerable<TSource>, Action<TSource>)

Executes a side-effect on every element of a sequence.

public static IEnumerable<TSource> Do<TSource>(this IEnumerable<TSource> source, Action<TSource> sideEffect)

Parameters

source IEnumerable<TSource>

An IEnumerable<T> sequence.

sideEffect Action<TSource>

An action with side-effects.

Returns

IEnumerable<TSource>

The original source sequence.

Type Parameters

TSource

The type of the elements of the source sequence.

Remarks

The sideEffect action is executed on each element yielded by the source sequence, immediately before the element is in turn yielded by this method.

This method is semantically equivalent to the following standard LINQ expression: 

source.Select(element => { sideEffect(element); return element; })
